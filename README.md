# How to Create a ServiceNow Developer Account

This guide walks you through creating a free ServiceNow Developer
account and obtaining your own Personal Developer Instance (PDI) for
learning and practice.

## Prerequisites

-   A valid email address
-   A modern web browser (Chrome, Edge, Firefox, or Safari)

------------------------------------------------------------------------

# Step 1: Go to the ServiceNow Developer Portal

Visit:

https://developer.servicenow.com/

Click **Sign In**.

------------------------------------------------------------------------

# Step 2: Create an Account

1.  Click **Create Account**.
2.  Enter:
    -   First Name
    -   Last Name
    -   Email Address
    -   Username
    -   Password
3.  Accept the terms.
4.  Click **Create Account**.

------------------------------------------------------------------------

# Step 3: Verify Your Email

Check your inbox for a verification email from ServiceNow.

Click the verification link to activate your account.

------------------------------------------------------------------------

# Step 4: Sign In

Return to the Developer Portal and sign in with your new credentials.

------------------------------------------------------------------------

# Step 5: Request a Personal Developer Instance (PDI)

After signing in:

1.  Click **Develop**.
2.  Select **Instances**.
3.  Click **Request Instance**.
4.  Choose the latest available release (for example, Washington DC,
    Xanadu, or the current release).
5.  Click **Request**.

Provisioning typically takes a few minutes.

------------------------------------------------------------------------

# Step 6: Open Your Instance

When the instance is ready, click **Open Instance**.

You'll receive: - Instance URL - Username (usually **admin**) -
Temporary password

Change the password after your first login.

------------------------------------------------------------------------
